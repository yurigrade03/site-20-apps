Atividade 20 - Jogo de Memória

Descrição: Jogo interativo de memória com cartas que devem ser viradas para encontrar pares iguais. O tabuleiro apresenta 16 cartas (8 pares), que são embaralhadas no início. O usuário clica nas cartas para revelá-las, tentando lembrar a posição para encontrar os pares correspondentes.

Implementação com HTML e CSS para estrutura e estilo do tabuleiro e cartas, e JavaScript para lógica do jogo, incluindo embaralhamento, controle de estado e verificação de pares. Data realizada: 01/06/2025

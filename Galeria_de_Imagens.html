Galeria de Imagens com Filtro

Descrição: Construa uma galeria de imagens com categorias (ex: natureza, tecnologia, pessoas). Crie botões para filtrar e mostrar apenas imagens da categoria selecionada.

Ultizando um visual minimalista na pagina principal, onde no js foi criado as variaveir para puxar as funções dos botões, e criado uma classificação na própria imagens, assim categorizando elas, para um laço no JS, fazendo com que ele aplique um display block nas imagens nao selicionadas no filtro para ocultar elas.

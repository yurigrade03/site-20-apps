Atividade 11 - Simulador de Caixa Eletrônico

Descrição: Página interativa onde o usuário informa o valor que deseja sacar e o sistema calcula a quantidade de notas necessárias para compor esse valor, utilizando notas de R$100, R$50, R$20, R$10, R$5 e R$2. Caso o valor não possa ser sacado exatamente (por falta da nota de R$1), o sistema informa o restante que não pode ser sacado.

Implementação com foco em lógica de divisão e resto, usando JavaScript para interação dinâmica. Data realizada: 23/05/2025

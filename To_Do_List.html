Lista de Tarefas

    To-Do List

    Descrição: Crie uma lista de tarefas onde o usuário possa digitar uma nova tarefa em um campo de texto e clicar em "Adicionar". A tarefa deve aparecer na lista abaixo com opções para marcar como concluída (riscar o texto) e para remover a tarefa.

    Uma atividade realizada com um layout simples, com apenas um input para digitação, com uma criação de uma lista ordenada através do JS, adiçao de dois botoes junto com a criaçao da lista com as funções de riscar a lista e remover uma tarefa. Data realizada: 23/05/2025

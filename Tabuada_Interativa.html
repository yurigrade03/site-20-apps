Atividade 03 - Tabuada Interativa

Descrição: Crie uma página web interativa onde o usuário pode digitar um número e, ao clicar em "Gerar Tabuada", o sistema calcula e exibe a tabuada completa desse número de 1 a 10. O layout utiliza HTML, CSS para o estilo, e JavaScript para gerar dinamicamente os resultados da tabuada na mesma página, sem recarregar.

Interface simples e funcional, com foco em usabilidade e clareza na apresentação dos resultados. Data realizada: 23/05/2025

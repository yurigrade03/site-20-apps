Atividade 08 - Verificador de Números Primos

Descrição: Página interativa que permite ao usuário digitar um número inteiro e verificar se ele é primo. O sistema realiza a verificação usando JavaScript, testando os divisores até a raiz quadrada do número, e apresenta o resultado na tela com mensagens claras para entradas inválidas e resultados.

Interface simples e objetiva, facilitando o entendimento do conceito de números primos. Data realizada: 23/05/2025

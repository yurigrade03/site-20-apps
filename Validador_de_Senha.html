Atividade 16 - Validador de Senha Forte

Descrição: Página interativa onde o usuário digita uma senha e, ao clicar no botão, o sistema valida se a senha atende a critérios de segurança: mínimo 8 caracteres, letras maiúsculas e minúsculas, números e caracteres especiais. Exibe os resultados em tempo real.

Implementação com HTML para a interface, CSS para o estilo e JavaScript para validar a senha com expressões regulares e alternar a visibilidade da senha. Data realizada: 01/06/2025
